# Calculator
Simple  calculator for Android. Solve some problem algebra and calculus.
in this calculator,  develop the following feature.

1. Mathematical operator: Addition (+),
Subtraction (-), Multiplication (*),
Division (/).
2. Digits: 1,2,3,4,5,6,7,8,9,0.
3. Display Screen: display the
mathematical expression and
The result.
4. Clear Screen Button.
5. Calculate button (=).
